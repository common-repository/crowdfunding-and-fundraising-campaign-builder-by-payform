Add a crowdfunding campaign to any Wordpress website in seconds, connected to Stripe or PayPal, using Crowdfunding for PayForm

== Description ==

PayForm is a solution used by hundreds of businesses and organizations around the world. The crowdfunding widget is perfect for anyone who needs to run a crowdfunding campaign on their Wordpress site. PayForm gives you the most powerful billing and order management solution to make your crowdfunding campaign a success.

Accept all major credit cards, like Visa, MasterCard, American Express, Discover, JCB and Diners Club in your Wordpress site using PayForm.

>**PayForm works out of the box in all Wordpress sites. You don’t need any other plugin or to have SSL in your server.**

== Installation ==
This plugin requires the PayForm plugin. If you don’t have it, it will be installed automatically. After installing, just go to any post or page editor and a button called *Add PayForm* will appear above, next to Add media. Click it to add a crowdfunding form.
